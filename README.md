## Rogex Laboratories

**Rogex Laboratories** is an independent research software and neurotechnology lab focused on building serious experimental tools with clear scientific limits.

The current public focus is:

### PRISMA 3

**Subject-normalized EEG decoding software for research.**

PRISMA 3 explores a central idea:

> EEG should not be interpreted only through universal thresholds.  
> Each subject should be interpreted relative to their own baseline, variability and uncertainty profile.

Current public validation includes:

| Area | Status |
|---|---|
| EEG preprocessing | Implemented |
| Feature extraction | Implemented |
| Personal baseline modelling | Implemented |
| Individual translator | Implemented |
| Synthetic supervised demo | Implemented |
| Real EEG ingestion | Implemented |
| OpenNeuro real EEG validation | Initial validation complete |
| Clinical/medical use | Not claimed |

PRISMA 3 is **not a medical device** and does **not** provide diagnosis.

---

## Current PRISMA 3 real EEG validation

Initial public validation on real EEG data:

| Metric | Result |
|---|---:|
| Dataset | OpenNeuro DS007358 |
| Task | Eyes closed vs eyes open |
| Subjects | 28 |
| Windows | 3304 |
| Raw features LOSO accuracy | 71.5% |
| Subject-normalized LOSO accuracy | 84.2% |
| Personalized intra-subject CV | 91.4% |
| Gain from normalization | +12.7 pts |

The key result is not only the final accuracy.

The important point is that **subject normalization improves generalization across people**.
